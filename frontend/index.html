Запрос: Текст, для которого требуется найти схожие по описанию запросы. Лимит: Ограничение на количество найденных совпадений в порядке убывания. Альфа: Коэффициент балансировки (0–1).
α = 0 — используется только NN.
α = 1 — используется только BM25. Продукт: Название продукта, в коллекции которого будет осуществляться поиск Фильтры для поиска запросов:
Клиент Полное наименование клиента в SD. Дата с Дата завершения запроса от. Дата по Дата завершения запроса до.
Быстрый поиск: True — быстрый поиск по индексам.
False — Полный поиск по всей коллекции.
